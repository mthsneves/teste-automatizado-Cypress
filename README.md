Projeto de testes automatizados utilizando Cypress para validação de cadastro, login e requisições de API.
Os testes estão organizados na pasta cypress/e2e e utilizam massa de dados quando necessário.

Como executar:

Instalar dependências com o comando: npm install

Executar o Cypress em modo visual: npx cypress open

Executar os testes em modo headless: npx cypress run

Cenários incluídos:

Cadastro (correto, incorreto e com massa de dados)

Login (correto e incorreto)

Testes de API
